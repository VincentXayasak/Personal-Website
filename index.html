Projects

Michelin Restaurant Data Search

This program is designed to scrape restaurant data from the Michelin Guide website for a specified location and present it in an easy-to-navigate graphical user interface (GUI). Users can search for restaurants by city or cuisine, view detailed information, and visit the restaurant's webpage directly. The data is stored in a local JSON file and SQLite database to avoid repeated web scraping, ensuring quick access to restaurant details.

National Park Finder

This program is a graphical application that allows users to search for national parks across the United States by selecting up to five states. It retrieves park data from the National Park Service API, including park names, descriptions, available activities, and related information. The user can then choose specific parks to save their information into JSON files, which are organized by state. The program features a dynamic interface that updates in real-time, displaying the results of API calls and enabling users to store park details for later use.

Bay Area Rainfall Watershed Data Search

This program provides an interactive tool for managing and visualizing precipitation data from various watersheds. Through a graphical user interface, users can select specific watersheds and time ranges to view detailed precipitation information, access full data for individual watersheds, and rank watersheds or sensors based on their precipitation levels over selected periods.
